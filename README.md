# python-sales-report
Python project for sales data analysis and Excel reporting

### Description
This project reads a CSV file and generates an Excel report using Python. The report includes:

- Total sales calculation (quantity × price)
- Sales analysis per region
- Formatted Excel table
- Chart visualization of sales per region

### Tools used
- Python
- openpyxl
- CSV

### How to run
Run main.py to generate the Excel report.
